Техническое задание от IDEADIGITAL .Agency

Задача: верстка и адаптив под мобильные устройства страницы по макету в figma

Макет в Figma
Видео

Header
В горизонтальное меню выведена навигация по странице, логотип и контактный номер телефона. Header имеет прозрачность Opacity: 80% Header прилипает и скролится по странице, при отображении меню начиная со 2 блока страницы прозрачность Header исчезает, он стает белым и появляется тень. Изменение размеров и/или удаление одного из блоков области header не должно влиять на другой блок. Редактирование/удаление/добавление пунктов не должно влиять на выравнивание. Номер телефона в области header должен быть кликабельным.
1 экран: Промо
Видео, оно циклично. На видео есть Заголовок и подзаголовок. Видео первоначально запускается без звука, при этом кнопка в состоянии выкл, при клике на кнопку звук запускается и кнопка меняет свое отображение на состояние вкл (громкоговоритель с звуковыми волнами)
2 экран: Отзывы
На этом экране отзывы, они перелистываются с помощью слайдера. Активный отзыв-белый уходит влево, отзыв который был справа становится по центру и стает белого цвета. Высота превью отзыва фиксированная. Активный отзыв 540*620, превью неактивного отзыва 400*370. Если количество текста не помещается в фиксированную область появляется кнопка Читать еще При клике на нее отзыв разворачивается, отображение развернутого отзыва вынесено в figma рядом с блоком Отзывы. Кнопка меняет название на Свернуть Ограничить число точек для пролистывания слайдера 3мя точками
3 Экран: Форма связи
Форма содержит 4 поля. Поля Имя, Номер телефона, Эл почта -обязательны к заполнению. Обязательное поле, которое не заполнено или заполнено с ошибкой, должно быть стилизовано. Такое отображение оно имеет только после клика на кнопку Отправить, если оно не заполнено или заполнено с ошибкой
Футер
Ссылки на соц сети, при клике должны открываться в новых окнах на десктопе и в мобильной версии Fb тоже в новой вкладке Fb https://www.facebook.com/ideadigitalagency/ Instagram https://www.instagram.com/ideadigitalagency/
Адаптив
Видео на 1 экране сохраняем, кнопку вкл/выкл звук тоже сохраняем. Остальные блоки перестраиваются согласно дизайну в figma

HTML верстка для IDEADIGITAL.Agency выполнена по БЭМ методологии, блочной версткой, с использованием flexbox и jQuery.
Время работы ~3-5 часов
