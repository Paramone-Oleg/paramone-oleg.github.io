Идея шаблона

Написать и создать HTML верстку по макету, адаптировать ее под CMS uCoz, сделать автоматическим добавление фильмов и максимально удобным для пользователей

«WFilms - Bootstrap 4 киношаблон для uCoz» - для чего же он создан? Этот шаблон идеально подойдет тем кому надоели типичные киношаблоны , для тех кто не ставит для пользователей рамки, кто хочет красиво оформить киносайт, или сайт человека который выпускает видео продукт, видеороликов блогера, медиа студии, фотографов. С помощью минималистического, не навязчивого дизайна вы можете эффективно и привлекательно презентовать свои работы аудитории и клиентам, а юзабилити и "интуитивность" поможет пользователям и гостям сайта комфортно находится на сайте.
Первый дизайн с Bootstrao 4 на uTemplate который ломает типичное рамки для подобных сайтов

Заметка: Изображения и описания к материалам, используемые в демо, не включены в архив - для ограничений лицензии. Они заменяются фиктивными заполнителями.

*Автодобавление - автозаполнение некоторых полей, взято с API плеера.
*Кредиты - плагины присутствуют или отсутствуют в зависимости от нужности в адаптации\верстки.
*Интеграция - под интеграцией подразумевается способность работы плеера на сайте. Пример для YouTube: возможность вставлять каналы, плейлисты, брать видео с 4 видов ссылок.
